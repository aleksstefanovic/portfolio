Smart Home Prototype

One of the earliest prototypes of what I now refer to as Pandora, Yeni (Youthful Energetic Natural Intelligence) was a mishmash different scripts. Yeni could read the temperature and respond to basic questions similar to Pandora. Some key differences included that Yeni was a local piece of software that ran off a usb, and she was primarily coded in a weird combination of C (for arduino) and Visual Basic. Yeni also used a specific text-to-speech API, known as Ivona. The eventual goal of Yeni was to be able to control parts of a home, through hardware and mechanical devices connected through WiFi.


Utility Glove

The utility glove project was one of my attempts to make something solely using arduino. The concept was a glove that a user wore, which provided a number of different sensors of the nearby environment and displayed that data onto a connected LCD screen on the user's wrist. The project was abandoned due to its lack of useful practical applications.

Retro Hardware Video Game

My only project, done with a friend, that was entirely done in hardware. Using the verilog language, and a Field Programmable Gate Array (FPGA), we built a very basic retro style video game but entirely from the hardware up. This included the design and coding of logic gates for functionality, flip-flops for memory, and a custom VGA adapter for displaying pixels. Smartphone Computer

A project currently in development, it revolves around creating an adapter that can turn a smartphone (in this case an LG G5) into a fully functional laptop/desktop replacement. The phone provides memory, cpu, video, and hard disk storage. Running any distribution of linux, all the user need do it plug in a keyboard of any kind and a monitor, and they can perform work they normally do on any normal machine. With the constant increase in the power of smartphones, this is becoming a very valid option in the near future. Currently only prototype is done, the second stage being building a laptop like "shell" which the phone can plug into to give the user a full keyboard and monitor to use.
