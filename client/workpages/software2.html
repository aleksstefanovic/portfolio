Pandora Virtual Assistant

A hobby project, Pandora is a simple cloud based virtual assistant built on nodejs. Hosted on the free service Heroku , she is comprised by three major components. A Postgres database provides storage of data, an express web server to host the REST API, and the actual API itself written in nodejs. To interface with Pandora, I have built a simple android app with voice recognition that will make the appropriate web service calls to the site to interpret the user's query. Currently she only knows a few basic commands. In addition to the above, I have built into her the ability to respond with the current temperature of my home. This was done with a very basic circuit, combining a temperature sensor and an arduino to transmit the data to a local PC, which updates Pandora's Postgres database. Video Game "sudo"

A hobby project currently in development, it is a deceptively simple three-dimensional realtime strategy game for PC. With minimalistic graphics, a huge amount of variety and strategy, as well as fast-paced multi-task heavy gameplay, sudo (named after "super user do" in linux systems) is meant to be an evolution of the genre pioneered by games such as Starcraft, Warcraft, Age of Empires, etc. It is being developed in Unity, using a mixture of built in tools and C#. An unfinished (and unplayable) prototype can be built from source below. This project is the reincarnation of a number of other failed game related projects done in CryEngine, Unreal Engine, and Starcraft 2 Game Editor.
